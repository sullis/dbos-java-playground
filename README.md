# dbos-java-playground

A Java playground for experimenting with [DBOS](https://dbos.dev) concepts.

## Requirements

- Java 21 ([Eclipse Temurin](https://adoptium.net) recommended)
- Maven

## Build & Test

```bash
# Compile and run tests
mvn clean package

# Run tests only
mvn clean test
```
